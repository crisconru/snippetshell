# Fecha y Hora

Para ajustar la fecha usamos el comando `date`

## Fecha

`sudo date +%Y%m%d -s "20170628"`

## Hora (formato 24 horas)

`sudo date +%T -s "12:47:24"`

## Hora (formato 12 horas)

`sudo date +%T%p -s "12:47:24AM"`  
`sudo date +%T%p -s "12:47:24PM"`
